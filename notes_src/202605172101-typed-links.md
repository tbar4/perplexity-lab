# Typed links

The Zettelkasten supports eight typed relationships between notes, plus
a default `related` edge:

| Type | Cluster | Meaning |
| --- | --- | --- |
| `references` | evidential | cites another note (usually a source / literature note) |
| `supports` | evidential | argues *for* a claim |
| `contradicts` | evidential | argues *against* a claim |
| `example_of` | evidential | concrete instance of an abstract idea |
| `elaborates` | structural | expands or refines a parent idea |
| `defines` | structural | this note defines this term |
| `derived_from` | structural | provenance / lineage |
| `questions` | interrogative | raises an open question about |
| `related` | associative | default; weak unstructured association |

Every edge is unidirectional in declaration but bidirectional in the
backlinks index — note B's page shows the inverse (e.g. "supported by →
note A").
